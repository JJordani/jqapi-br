.not()

Categorias: Traversing > Filtering

.not( seletor ) Retorna: jQuery

Descrição: Remove elementos do conjunto de elementos selecionados.

  • versão adicionada: 1.0 .not( seletor )

    seletor Uma string contendo uma expressão seletora para coincidir os elementos de acordo com ela.

  • versão adicionada: 1.0 .not( elementos )

    elementos Um ou mais elementos DOM para serem removidos do conjunto de elementos selecionados.

  • versão adicionada: 1.4 .not( function(index) )

    function(index) Uma função usada como um teste para cada elemento no conjunto. this é o elemento DOM atual.

Dado um objeto jQuery que representa um conjunto de elementos DOM, o método .not() constroi um novo objeto jQuery a partir de um subconjunto dos elementos correspondentes. O seletor fornecido é testado em comparação a cada elemento; os elementos que não corresponderem com o seletor serão incluídos no resultado.

Considere uma página com uma lista simples:

<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li>list item 3</li>
  <li>list item 4</li>
  <li>list item 5</li>
</ul>

Podemos aplicar este método para o conjunto de itens da lista:

$('li').not(':even').css('background-color', 'red');

O resultado dessa chamada é um fundo vermelho para os itens 2 e 4, uma vez que não correspondem ao seletor (lembre-se que: :even e :odd usam indexação base 0).

Removendo Elementos Específicos

A segunda verão do método .not() nos permite remover elementos do conjunto correspondente, supondo que tenhamos encontrado os elementos anteriormente por outros meios. Por exemplo, suponha que a nossa lista tinha um ID aplicado em um de seus itens:

<ul>
  <li>list item 1</li>
  <li>list item 2</li>
  <li id="notli">list item 3</li>
  <li>list item 4</li>
  <li>list item 5</li>
</ul>

Podemos buscar o terceiro item da lista usando a função getElementById() nativa do JavaScript, e então remove-lo do objeto jQuery:

$('li').not(document.getElementById('notli'))
  .css('background-color', 'red');

Esta instrução altera a cor dos itens 1, 2, 4 e 5. Poderíamos ter feito a mesma coisa com uma expressão jQuery mais simples, mas esta técnica pode ser útil quando, por exemplo, outras bibliotecas fornecerem referências para nós DOM simples.

A partir do jQuery 1.4, o método .not() pode ter uma função como argumento da mesma maneira que o .filter() possui. Elementos para a qual a função retorna true são excluídos do conjunto filtrado; todos os outros elementos são incluídos.

Exemplo: Adiciona uma borda às divs que não estão verdes ou azuis. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:50px; height:50px; margin:10px; float:left;
        background:yellow; border:2px solid white; }
  .green { background:#8f8; }
  .gray { background:#ccc; }
  #blueone { background:#99f; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>
  <div id="blueone"></div>
  <div></div>
  <div class="green"></div>

  <div class="green"></div>
  <div class="gray"></div>
  <div></div>
<script>
    $("div").not(".green, #blueone")
            .css("border-color", "red");

</script>

</body>
</html>

Exemplo: Remove o elemento com o ID "selected" do conjunto de todos os parágrafos. 

$("p").not("#selected")

Exemplo: Remove todos os elementos que combinam com "div p.selected" a partir do conjunto total de todos os parágrafos. 

$("p").not($("div p.selected"))
