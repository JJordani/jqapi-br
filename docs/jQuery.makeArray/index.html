jQuery.makeArray()

Categorias: Utilities

jQuery.makeArray( obj ) Retorna: Array

Description: Converter um objeto array-like em um autentico Array Javascript.

  • adicionado na versão: 1.2jQuery.makeArray( obj )

    objAlgum objeto para converter em um Array nativo.

Muitos métodos, tanto em jQuery como JavaScript em geral, são objetos de retorno array-like. Por exemplo, a função de seletor jQuery $() retorna um objeto jQuery que tem muitas das propriedades de uma array (um tamanho, o [] operador de acesso de array, etc), mas não é exatamente o mesmo como uma matriz e não possui alguns métodos internos de uma matriz (como .pop() e .reverse()).

Note-se que após a conversão, todas as características especiais que o objeto tinha (como os métodos jQuery em nosso exemplo) deixarão de estar presentes. O objeto é agora um array simples.

Exemplos:

Exemplo: Passar uma coleção de HTMLElements para uma Array. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>First</div>
  <div>Second</div>  
  <div>Third</div>

  <div>Fourth</div>
<script>
    var elems = document.getElementsByTagName("div"); // returns a nodeList
    var arr = jQuery.makeArray(elems);
    arr.reverse(); // use an Array method on list of dom elements
    $(arr).appendTo(document.body);
</script>

</body>
</html>
