event.target

Categories: Events > Event Object

event.target Retorna: Element

Description: O elemento DOM que inicializou o evento.

  • adicionado na versão: 1.0event.target

A propriedade target pode ser o elemento que registrou ele ou algum descendente dele. Muitas vezes é útil comparar event.target com this em ordem para determinar se o evento está a ser tratado devido a subida do evento. Esta propriedade é muito utilizada em casos de delegações, quanto em diversos eventos.

Exemplos:

Exemplo: Exibir o nome da tag no click 

<!DOCTYPE html>
<html>
<head>
  <style>
span, strong, p { 
  padding: 8px; display: block; border: 1px solid #999;  }
    </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div id="log"></div>
<div>
  <p>
    <strong><span>click</span></strong>
  </p>
</div>
<script>$("body").click(function(event) {
  $("#log").html("clicked: " + event.target.nodeName);
});  </script>

</body>
</html>

Exemplo: Implementar um simples evento de delegeção: O manipulador do click é adicionado a uma lista desordenada, e o filho desta li fica escondido. Clicando em um dos elementos li, seus herdeiros alternam (leia toggle()). 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<ul>
  <li>item 1
    <ul>
      <li>sub item 1-a</li>
      <li>sub item 1-b</li>
    </ul>
  </li>
  <li>item 2
    <ul>
      <li>sub item 2-a</li>
      <li>sub item 2-b</li>
    </ul>
  </li>  
</ul>
<script>function handler(event) {
  var $target = $(event.target);
  if( $target.is("li") ) {
    $target.children().toggle();
  }
}
$("ul").click(handler).find("ul").hide();</script>

</body>
</html>
